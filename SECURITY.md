# Security Policy

## Reporting a Vulnerability

Please report security issues privately. Do **not** open a public issue.

You can report a vulnerability in either of these ways:

- Open a private advisory via the repository's **Security** tab
  (Security → Advisories → Report a vulnerability).
- Email **babak@cocode.dk**.

You will receive an acknowledgement within **5 business days**. We will keep
you informed as we investigate and work toward a fix.

## Supported Versions

| Version        | Supported |
| -------------- | --------- |
| latest         | ✅        |
| older releases | ❌        |
